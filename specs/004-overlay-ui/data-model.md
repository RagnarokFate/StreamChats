# Data Model: Overlay UI

The UI consumes events generated by the `@obs-chat/event-schema` package.

## Component State

### ChatMessage State
```typescript
interface UIMessage {
  eventId: string;
  platform: 'twitch' | 'youtube' | 'custom';
  timestamp: string;
  author: {
    id: string;
    name: string;
  };
  text: string; // The fully processed/masked text
  isDeleted: boolean; // Flagged when a moderation event strikes
}
```
