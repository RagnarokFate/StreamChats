All Your Chats, One Overlay

Connect to the platforms your viewers use. Every message, every platform, one unified feed.

Twitch

IRC WebSocket connection with full badge and emote support.

YouTube

Live chat polling with support for Super Chats and memberships.

Kick

Real-time connection via Pusher WebSocket protocol.

TikTok

Live stream chat and gift events via protobuf WebSocket.
